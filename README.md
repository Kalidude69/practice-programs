## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Kalidude69/practice-programs.git
   ```
2. Navigate to the desired folder.
3. Run or compile the programs as needed:
   ```bash
   # Example for Python
   python filename.py
   ```

   ```bash
   # Example for C
   gcc filename.c -o output && ./output
   ```
